> Полный курс iOS разработки: от Swift до production-ready интернет-магазина на VIPER

## 🎯 О книге

Эта книга проведёт вас от **полного нуля** до создания **полноценного iOS-приложения** интернет-магазина. Вы изучите Swift, UIKit, архитектуру VIPER и современные практики iOS разработки.

**Финальный проект:** Приложение интернет-магазина с авторизацией, каталогом, корзиной, избранным и заказами.

## 🛠 Технологии

| **UIKit** | UI фреймворк |
| **SnapKit** | Programmatic Auto Layout |
| **Firebase** | Auth + Realtime Database |
| **Realm** | Локальное хранилище |
| **SDWebImage** | Загрузка изображений |
| **VIPER** | Архитектурный паттерн |

## 🚀 Быстрый старт

### Читать онлайн
```bash
# Клонировать репозиторий
git clone https://github.com/sirserik/mobile-ios-book.git
cd mobile-ios-book

# Открыть книгу
open html/index.html
```

### Запустить проект
```bash
# Клонировать исходный код
git clone https://github.com/BerkaySancar/ECommerce-VIPER.git

# Открыть в Xcode
open ECommerce.xcodeproj

# Установить зависимости через SPM (автоматически)
# Запустить на симуляторе (Cmd + R)
```
