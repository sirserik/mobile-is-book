Тесты на понимание

1. Что выведет этот код?

let x = 10
var y = x
y = 20
print(x)
10
20
Ошибка компиляции

Правильно!

В Swift Int - это value type (struct). При var y = x создаётся копия. Изменение y не влияет на x.

2. Какой тип у переменной name?

var name: String? = "John"
String
Optional<String> или String?
Any

Правильно!

String? - это синтаксический сахар для Optional<String>. Значение может быть либо String, либо nil.

3. Что делает @State в SwiftUI?

Делает переменную глобальной
Создаёт реактивное состояние, при изменении которого View перерисовывается
Сохраняет значение в UserDefaults

Правильно!

@State создаёт source of truth для View. При изменении значения SwiftUI автоматически перерисовывает View.

4. Что означает $text в SwiftUI?

@State private var text = ""
TextField("Введите", text: $text)
Значение переменной text
Binding к переменной text (двусторонняя связь)
Шаблонная строка (string interpolation)

Правильно!

$ создаёт Binding - двустороннюю связь между @State и компонентом. TextField может как читать, так и изменять значение.

5. Что произойдёт при выполнении этого кода?

let user: User? = nil
print(user!.name)
Выведет nil
Выведет пустую строку
Приложение упадёт (crash)

Правильно!

Force unwrap (!) вызывает crash если значение равно nil. Используйте if let, guard let или ?? вместо !

6. В чём разница между struct и class?

struct - value type (копируется), class - reference type (ссылка)
struct быстрее, class медленнее
Никакой разницы, это синонимы

Правильно!

При присваивании struct копируется целиком, а class передаётся по ссылке. Изменение копии struct не влияет на оригинал.

Задача 1: Безопасное извлечение

Перепишите код так, чтобы он не падал при nil: 

func greet(name: String?) {
    print("Hello, \(name!)")
}
// Вариант 1: if let
func greet(name: String?) {
    if let name = name {
        print("Hello, \(name)")
    } else {
        print("Hello, Guest")
    }
}

// Вариант 2: nil coalescing
func greet(name: String?) {
    print("Hello, \(name ?? "Guest")")
}

// Вариант 3: guard
func greet(name: String?) {
    guard let name = name else {
        print("Hello, Guest")
        return
    }
    print("Hello, \(name)")
}

Задача 4: ViewModel для списка товаров

Создайте @Observable ViewModel для загрузки товаров с API. Должны быть: массив products, флаг isLoading, метод loadProducts(). 

@Observable
class ProductsViewModel {
    var products: [Product] = []
    var isLoading = false
    var errorMessage: String?

    func loadProducts() async {
        isLoading = true
        errorMessage = nil

        do {
            let url = URL(string: "https://dummyjson.com/products")!
            let (data, _) = try await URLSession.shared.data(from: url)
            let response = try JSONDecoder().decode(ProductsResponse.self, from: data)
            products = response.products
        } catch {
            errorMessage = error.localizedDescription
        }

        isLoading = false
    }
}

struct ProductsResponse: Codable {
    let products: [Product]
}

// Использование во View
struct ProductsView: View {
    @State private var viewModel = ProductsViewModel()

    var body: some View {
        List(viewModel.products) { product in
            Text(product.title)
        }
        .overlay {
            if viewModel.isLoading {
                ProgressView()
            }
        }
        .task {
            await viewModel.loadProducts()
        }
    }
}

Задача 5: Корзина покупок

Создайте CartManager с методами add, remove, clear и computed property total. 

@Observable
class CartManager {
    var items: [CartItem] = []

    var total: Double {
        items.reduce(0) { $0 + $1.product.price * Double($1.quantity) }
    }

    var itemCount: Int {
        items.reduce(0) { $0 + $1.quantity }
    }

    func add(_ product: Product) {
        if let index = items.firstIndex(where: { $0.product.id == product.id }) {
            items[index].quantity += 1
        } else {
            items.append(CartItem(product: product, quantity: 1))
        }
    }

    func remove(_ product: Product) {
        if let index = items.firstIndex(where: { $0.product.id == product.id }) {
            if items[index].quantity > 1 {
                items[index].quantity -= 1
            } else {
                items.remove(at: index)
            }
        }
    }

    func clear() {
        items.removeAll()
    }
}

struct CartItem: Identifiable {
    let id = UUID()
    let product: Product
    var quantity: Int
}
