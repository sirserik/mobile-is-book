Архитектура MVVM

MVVM (Model-View-ViewModel) — это архитектурный паттерн, который отлично подходит для SwiftUI благодаря реактивной природе фреймворка. 

┌─────────────────────────────────────────────────┐
│                      MVVM                       │
├─────────────────────────────────────────────────┤
│                                                 │
│   ┌──────────┐         ┌─────────────┐         │
│   │   VIEW   │ ◄─────► │  VIEWMODEL  │         │
│   │ (SwiftUI)│         │ (@Observable)│         │
│   └──────────┘         └──────┬──────┘         │
│                               │                 │
│                               ▼                 │
│                        ┌─────────────┐         │
│                        │    MODEL    │         │
│                        │  (Structs)  │         │
│                        └─────────────┘         │
│                                                 │
│   View: Отображение UI                         │
│   ViewModel: Бизнес-логика, состояние          │
│   Model: Данные (Product, Cart, etc.)          │
│                                                 │
└─────────────────────────────────────────────────┘
