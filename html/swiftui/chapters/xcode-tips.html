Горячие клавиши и трюки для продуктивной работы

Горячие клавиши

Самые важные

Cmd + R Run Запустить приложение
Cmd + B Build Скомпилировать проект
Cmd + . Stop Остановить выполнение
Cmd + Shift + K Clean Build Очистить кэш сборки

Навигация

Cmd + Shift + O Open Quickly Быстрый поиск файла
Cmd + Shift + J Reveal in Navigator Показать файл в дереве
Ctrl + 6 Jump to Method Список методов в файле
Cmd + L Go to Line Перейти к строке
Cmd + Click Jump to Definition Перейти к определению
Cmd + Ctrl + ← Go Back Вернуться назад

Редактирование

Cmd + / Toggle Comment Закомментировать строку
Cmd + D Duplicate Line Дублировать строку
Cmd + [ / ] Indent Изменить отступ
Ctrl + I Re-Indent Исправить отступы
Option + ↑ / ↓ Move Line Переместить строку
Cmd + Option + [ / ] Fold/Unfold Свернуть/развернуть блок

Отладка

Cmd + \ Toggle Breakpoint Поставить/убрать точку останова
Cmd + Y Enable/Disable Breakpoints Включить/выключить все
F6 Step Over Следующая строка
F7 Step Into Войти в функцию
F8 Step Out Выйти из функции
Ctrl + Cmd + Y Continue Продолжить выполнение

SwiftUI Preview

Открыть/закрыть Preview

Cmd + Option + Enter

Показывает или скрывает Canvas с превью справа от редактора.

Обновить Preview

Cmd + Option + P

Перезапускает превью если оно зависло или показывает ошибку.

Live Preview

Нажмите кнопку Play на превью или используйте Cmd + Option + P

Включает интерактивный режим - можно кликать на кнопки и вводить текст.

Preview на разных устройствах

#Preview {
    ContentView()
}

#Preview("iPhone SE") {
    ContentView()
        .previewDevice("iPhone SE (3rd generation)")
}

#Preview("Dark Mode") {
    ContentView()
        .preferredColorScheme(.dark)
}

Советы

Используйте Cmd + Shift + O

Это самый быстрый способ найти файл или символ. Начните печатать название и выберите из списка.

Clean Build при странных ошибках

Если Xcode показывает странные ошибки, попробуйте Cmd + Shift + K для очистки кэша сборки.

Перезапустите Xcode

Если что-то не работает (Preview не обновляется, автодополнение сломалось) - перезапустите Xcode.

Derived Data

При серьёзных проблемах: удалите папку ~/Library/Developer/Xcode/DerivedData и перезапустите Xcode.
